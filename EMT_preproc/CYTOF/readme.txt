https://community.cytobank.org/cytobank/experiments#project-id=1296

 - PhEMD-E inhibitor dataset - Replicate 1
 - PhEMD-E inhibitor dataset - Replicate 2
 - PhEMD-E inhibitor dataset - Replicate 3
 - PhEMD-Multi-batch analyses-Inhibitor Dataset A
 - PhEMD-Multi-batch analyses-Inhibitor Dataset B
 - PhEMD-Multi-batch analyses-Inhibitor Dataset C
 - PhEMD-Multi-batch analyses-Inhibitor Dataset D
 - PhEMD-Multi-batch analyses-Inhibitor Dataset E
